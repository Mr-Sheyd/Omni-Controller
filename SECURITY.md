# Security Policy

## Supported Versions
Currently, only the latest release is actively supported with security updates.

| Version | Supported          |
| ------- | ------------------ |
| v1.0.x  | ✅ Yes              |
| < v1.0  | ❌ No               |

## Reporting a Vulnerability
If you discover a security vulnerability within this project, please **do not open a public Issue**. 

Instead, please report it directly via Telegram: [@Mr_Sheyd]
